
# 🏷️ Web Scraper - E-Commerce Laptops Data

A professional, simple, and clean Python web scraper built using **BeautifulSoup** and **Pandas** to extract laptop data from the WebScraper.io test e-commerce website.

---

## 🚀 Features

- **Data Extracted**: Product Title, Complete Link, Image URL, Price, Description, and Review Counts.
- **Robustness**: Handles multi-class elements and automatically resolves relative URLs into clickable links.
- **Output**: Saves data into a clean, well-structured `laptops_data.csv` file.

---

## 📁 File Structure

- `scraper.py` - The main Python source code for the scraper.
- `requirements.txt` - Required python libraries for this project.
- `README.md` - Project documentation (this file).
